# **La liste des commandes Unix et leur équivalent en PowerShell**

1. cp = Copy-Item
2. rm = Remove-Item
3. cd = Set-Location
4. mkdir = New-Item
5. man = Help
6. history = Get-History
7. alias = Set-Alias
8. cat = Get-Content
